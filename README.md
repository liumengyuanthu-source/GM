# General Mills Three-Journey AI Scenario Navigator

This repository contains the static HTML demo for the General Mills AI scenario case sharing navigator.

## Current Build

- Version: fixed3_3 interaction build with complete images
- Entry file: `index.html`
- Format: static HTML/CSS/JavaScript with local image assets

## What Is Included

- Customer Journey, Employee Journey, and Business Operation Journey navigation
- Journey map with zoom-focus drill-down interaction
- Persona segment matrix with scenario detail panels
- Benchmark case cards and NTT DATA capability references
- Bilingual Chinese / English presentation experience
- Local image assets generated and curated for the case story

## How To View

Open `index.html` directly in a browser, or publish this repository with GitHub Pages.

## Notes

This is a discussion and demo artifact for scenario exploration. It does not require a backend service and does not store scoring or user input data.
